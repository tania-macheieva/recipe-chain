# Recipe Chain (Node.js + Express + PostgresSQL)

Простий додаток для створення/редагування/видалення рецептів.

## Запуск локально
1. Клонувати репозиторій:
   ```bash
   git clone <your-repo-url>
   cd recipe-chain

2. Встановити залежності:

    npm install


3. Створити .env в корені (скопіювати з .env.example) і заповнити:

   PORT=5000



5. Запустити:

    npm run dev
